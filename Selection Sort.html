選擇排序法介紹

選擇排序法（Selection Sort）是一種簡單的排序算法。它的工作原理是每次從未排序的部分中選擇最小的元素，然後將其與未排序部分的首元素交換位置。

工作原理

選擇排序法將陣列分為已排序和未排序兩部分。初始時，已排序部分為空。每次從未排序部分中找到最小的元素，將其移到已排序部分的末尾。

步驟示例

假設我們有一個陣列：[5, 3, 8, 4, 2]

  • 第一步：找到最小元素 2，與第一個元素 5 交換： [2, 3, 8, 4, 5]
  • 第二步：從剩餘元素中找到最小元素 3，已在正確位置： [2, 3, 8, 4, 5]
  • 第三步：從剩餘元素中找到最小元素 4，與第三個元素 8 交換： [2, 3, 4, 8, 5]
  • 第四步：從剩餘元素中找到最小元素 5，與第四個元素 8 交換： [2, 3, 4, 5, 8]

時間複雜度

  • 最佳情況：O(n²)
  • 最壞情況：O(n²)
  • 平均情況：O(n²)

優缺點

優點：

  • 簡單易懂，實現容易
  • 交換次數較少（最多 n-1 次）
  • 對於小規模數據效率尚可

缺點：

  • 時間複雜度為 O(n²)，對於大規模數據效率低
  • 不穩定排序（相同元素的相對位置可能改變）

JavaScript 實現示例

function selectionSort(arr) {
    for (let i = 0; i < arr.length - 1; i++) {
        let minIndex = i;
        for (let j = i + 1; j < arr.length; j++) {
            if (arr[j] < arr[minIndex]) {
                minIndex = j;
            }
        }
        if (minIndex !== i) {
            [arr[i], arr[minIndex]] = [arr[minIndex], arr[i]];
        }
    }
    return arr;
}

// 測試
let array = [5, 3, 8, 4, 2];
console.log(selectionSort(array)); // [2, 3, 4, 5, 8]

選擇排序法雖然效率不如一些高級排序算法，但由於其簡單性和交換次數少的特性，在某些情況下仍然有用。
