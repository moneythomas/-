什麼是堆積排序法？

堆積排序法（Heap Sort）是一種利用堆（Heap）資料結構的排序演算法。 它將陣列視為一個完全二元樹，並利用堆的特性來進行排序。 堆積排序法是一種原地排序演算法，空間複雜度為 O(1)。

演算法流程

  1. 將無序陣列建構成一個最大堆（Max Heap）。
  2. 將堆頂元素（最大值）與陣列最後一個元素交換。
  3. 縮小堆的大小，並重新調整堆，使其滿足堆的特性。
  4. 重複步驟 2 和 3，直到堆的大小為 1。

具體範例

假設數列為：[4, 10, 3, 5, 1]

  • 建構最大堆：將陣列調整為堆結構。
  • 交換根節點 10 與最後元素 1，得到 [1, 5, 3, 4, 10]，堆大小減 1。
  • 調整剩餘堆，交換根節點 5 與最後元素 4，得到 [1, 4, 3, 5, 10]。
  • 繼續調整，最終得到排序陣列 [1, 3, 4, 5, 10]。

時間與空間複雜度

  • 時間複雜度：最佳、平均、最壞情況皆為 O(n log n)
  • 空間複雜度：O(1)，原地排序，不需要額外空間

優缺點

優點

  • 時間複雜度穩定且高效
  • 空間複雜度低，不需要額外空間
  • 適合處理大型資料集

缺點

  • 不穩定排序演算法，相等元素的相對順序可能改變
  • 實作較為複雜，需要理解堆的結構
  • 對於小規模資料效率不如簡單排序

JavaScript 實作範例

function heapSort(arr) {
    const n = arr.length;

    // 建構最大堆
    for (let i = Math.floor(n / 2) - 1; i >= 0; i--) {
        heapify(arr, n, i);
    }

    // 逐一提取元素
    for (let i = n - 1; i > 0; i--) {
        // 將根節點與最後一個元素交換
        [arr[0], arr[i]] = [arr[i], arr[0]];

        // 重新調整堆
        heapify(arr, i, 0);
    }

    return arr;
}

function heapify(arr, n, i) {
    let largest = i;
    const left = 2 * i + 1;
    const right = 2 * i + 2;

    // 如果左子節點大於根節點
    if (left < n && arr[left] > arr[largest]) {
        largest = left;
    }

    // 如果右子節點大於根節點
    if (right < n && arr[right] > arr[largest]) {
        largest = right;
    }

    // 如果最大值不是根節點
    if (largest !== i) {
        [arr[i], arr[largest]] = [arr[largest], arr[i]];

        // 遞歸調整受影響的子樹
        heapify(arr, n, largest);
    }
}

// 測試
const array = [4, 10, 3, 5, 1];
console.log(heapSort(array)); // [1, 3, 4, 5, 10]
