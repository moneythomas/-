什麼是合併排序法？

合併排序法（Merge Sort）是一種經典的排序演算法，屬於分治法（Divide and Conquer）。 它先將待排序陣列不斷拆分成更小的子問題，然後再將排序後的子陣列逐步合併成最終結果。

演算法流程

  1. 如果陣列長度小於或等於 1，則視為已排序，直接返回。
  2. 將陣列分成兩個大致相等的子陣列。
  3. 遞迴地對左子陣列和右子陣列各自進行合併排序。
  4. 將兩個已排序的子陣列合併成一個完整的排序陣列。

具體範例

假設數列為：[5, 2, 8, 1, 6]

  • 拆分成 [5, 2] 與 [8, 1, 6]
  • 繼續拆分並排序成 [2, 5] 與 [1, 6, 8]
  • 合併兩個已排序陣列，得到最終結果 [1, 2, 5, 6, 8]

時間與空間複雜度

  • 時間複雜度：最佳、平均、最壞情況皆為 O(n log n)
  • 空間複雜度：O(n)，合併階段需要額外暫存陣列

優缺點

優點

  • 排序時間穩定且高效，適合大型資料集
  • 與原始資料的初始順序無關
  • 是一種穩定排序演算法，相等元素順序不會改變

缺點

  • 需要額外的暫存空間，空間複雜度為 O(n)
  • 遞迴實作較為複雜，不一定適合極端記憶體受限環境

JavaScript 實作範例

function mergeSort(arr) {
    if (arr.length <= 1) {
        return arr;
    }

    const mid = Math.floor(arr.length / 2);
    const left = mergeSort(arr.slice(0, mid));
    const right = mergeSort(arr.slice(mid));

    return merge(left, right);
}

function merge(left, right) {
    const merged = [];
    let i = 0;
    let j = 0;

    while (i < left.length && j < right.length) {
        if (left[i] <= right[j]) {
            merged.push(left[i]);
            i++;
        } else {
            merged.push(right[j]);
            j++;
        }
    }

    return merged.concat(left.slice(i)).concat(right.slice(j));
}

// 測試
const array = [5, 2, 8, 1, 6];
console.log(mergeSort(array)); // [1, 2, 5, 6, 8]
